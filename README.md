# Portfolio Website - IT103 FA1 Module 1

A minimalist personal portfolio website project for IT103 (FA1).

## About

This is my simple and clean portfolio website, designed to showcase who I am, my achievements, and my certifications. It includes a responsive layout and a dark/light theme toggle, making it easy to view on any device. The goal of this portfolio is to present my information in a straightforward and professional way while keeping the overall design minimal and easy to navigate.

## Project Structure

- `index.html` - Main HTML structure
- `style.css` - Styling and theme variables
- `script.js` - Theme toggle functionality
- `images/` - Images folder containing profile picture, project images, and certificate images

## Technologies

- HTML5
- CSS3
- JavaScript
